# Timesheet Calculator

Static browser app for calculating monthly working days in Portugal with:

- Portuguese national holidays
- Lisbon municipal holiday
- Multiple single-day or date-range days off
- Project-hour distribution capped at 8 hours per workday

## Usage

Open `index.html` in a browser and fill in:

1. Month and year
2. Days off
3. Project count and hours per project

Then click **Calculate timesheet**.
